## Calculadora em React ⚛️

Um projeto de calculadora simples desenvolvido com o objetivo de praticar e consolidar conhecimentos em desenvolvimento front-end utilizando React, TypeScript e Tailwind CSS.



## Sobre o projeto

Esse projeto foi feito com foco em:

- Praticar front-end básico
- Criar uma calculadora funcional
- Treinar lógica de programação


## O que foi usado:

- React
- TypeScript
- Tailwind CSS
- Vite
